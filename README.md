
# 🚚 Logistics Optimization for Delivery Routes (SQL)

## 📌 Project Overview

This project analyzes **logistics and delivery performance** for Amazon using SQL. The goal is to identify delivery delays, evaluate warehouse and delivery agent performance, and uncover inefficiencies in delivery routes.

By analyzing logistics data such as orders, routes, warehouses, and shipment tracking, the project generates **data-driven insights to improve route planning and delivery efficiency**. 

---

## 🎯 Objective

The objective of this project is to build a **SQL-based logistics analytics system** that:

* Identifies **delivery delays and their root causes**
* Analyzes **route efficiency and traffic delays**
* Evaluates **warehouse processing performance**
* Measures **delivery agent performance**
* Generates **KPIs to improve shipment efficiency**

---

## 📂 Dataset

The analysis uses the following datasets:

* **Orders** – Order details, delivery dates, and shipment information
* **Routes** – Route distance, traffic delays, and travel time
* **Warehouses** – Warehouse locations and processing details
* **Delivery Agents** – Delivery agent performance information
* **Shipment Tracking** – Shipment checkpoints and delay reasons

---

## 📊 Key Analysis

### 1️⃣ Delivery Delay Analysis

* Calculated delivery delay for each order
* Identified **top delayed routes**
* Ranked orders by delay within each warehouse

### 2️⃣ Route Optimization

* Calculated **average delivery time and traffic delays**
* Evaluated **distance-to-time efficiency ratio**
* Identified **routes with high delay percentages**

### 3️⃣ Warehouse Performance

* Identified warehouses with **highest processing time**
* Compared **total vs delayed shipments**
* Ranked warehouses by **on-time delivery percentage**

### 4️⃣ Delivery Agent Performance

* Ranked agents based on **on-time delivery rate**
* Identified **low-performing agents**
* Compared performance between **top and bottom agents**

### 5️⃣ Shipment Tracking Insights

* Identified **last shipment checkpoints**
* Analyzed **most common delay reasons**
* Detected orders with **multiple delay checkpoints**

---

## 📈 Key KPIs

* **On-Time Delivery %**
  `(Total On-Time Deliveries / Total Deliveries) × 100`

* **Average Delivery Delay per Region**

* **Average Traffic Delay per Route**
